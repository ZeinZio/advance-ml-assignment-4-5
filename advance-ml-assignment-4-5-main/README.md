# 🚢 Titanic Survival Predictor

Aplikasi Machine Learning untuk memprediksi keselamatan penumpang Titanic berdasarkan data seperti kelas tiket, jenis kelamin, jumlah saudara, tarif tiket, dan pelabuhan keberangkatan.

## 📋 Fitur Aplikasi

- Input data penumpang melalui form interaktif
- Prediksi apakah penumpang selamat atau tidak
- Menampilkan peluang keselamatan dalam persentase
- Tampilan yang responsif dan mudah digunakan

## 🛠️ Teknologi yang Digunakan

- Streamlit - Framework untuk membangun web app
- Scikit-learn - Library machine learning
- XGBoost - Algoritma klasifikasi
- Pandas & NumPy - Manipulasi data
- Joblib - Load model yang sudah dilatih
